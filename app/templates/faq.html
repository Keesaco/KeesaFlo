Frequently Asked Questions

Q1. What is KeesaFlo?

KeesaFlo is a cloud based web application designed to allow people to upload, analyse and share their flow cytometry data in an intuitive and user friendly manner.

Q2. Who are Keesaco?

Keesaco is a group of six students from the University of Bristol. They originally created this website for a group coursework.

Q3. Can I host my own version of KeesaFlo?

If you want to. Currently, the KeesaFlo website has been created to run using Google's developer services, so you will need to register a billing account with them. More information on setting up and running the site can be found in the KeesaFlo repository.

Q4. Can I make my own version of this website?

Sure. The KeesaFlo website is open source, so feel free to create your own derivative websites. For more information on our license, please visit the GNU GPL webpage.

Q5. Can I help develop the KeesaFlo website?

If you have a GitHub account, then feel free to contribute to the KeesaFlo repository on GitHub
